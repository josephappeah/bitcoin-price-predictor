SENTIMENT ANALYSIS Module

HOW IT WORKS


the sentiment analysis module works by combining the twitter developer api with the Valence aware dictionary and sentiment reasoner (vader) python library, to provide daily sentiment analysis on bitcoin.



tweets are collected on an hourly basis. after collection, the sentiment value of each tweet is obtained and normalized to range from 0 - 1. This is done using the VADER sentiment analysis tool. These values are averaged out per hour and then again for the daily value. All processing is done in python.
